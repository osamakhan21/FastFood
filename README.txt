# Fast Food Nutrition Portal 🍔🥤

A full-stack ERP-style Flask dashboard app with:
- Secure Login & Registration
- Dashboard with Summary Cards
- Nutrition Table with Search + Export to CSV
- Graphs (Bar & Pie Charts) using Chart.js
- Mobile Responsive UI with Bootstrap

---
